## 🎯 What is Skilo?

**Skilo** is a free, AI-powered career intelligence tool that analyzes your current skill set and tells you exactly what's missing to land your dream job — then builds a personalized weekly roadmap to get you there.

No more guessing what to learn. No more wasted time on irrelevant courses. Just a clear, direct path from where you are to where you want to be.

## ⚙️ How It Works

```
1. User uploads CV or pastes LinkedIn URL
         ↓
2. Skilo parses and extracts profile data
         ↓
3. Gemini AI analyzes skills vs job requirements
         ↓
4. Gap report + match score generated
         ↓
5. Personalized weekly roadmap created
         ↓
6. User tracks progress on dashboard
```

## 📦 Getting Started

### Option 1 — Use directly (No setup needed)
Just open `skilo.html` in any browser. That's it.

### Option 2 — Clone & run locally

```bash
# Clone the repository
git clone https://github.com/yourusername/skilo.git

# Navigate to project
cd skilo

# Open in browser
open skilo.html
```

### Option 3 — Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```

## 🔑 API Configuration

Skilo uses **Google Gemini API** for AI analysis.

1. Get your free API key at [aistudio.google.com](https://aistudio.google.com)
2. Open `skilo.html`
3. Find this line:
```javascript
const GEMINI_KEY = 'your_api_key_here';
```
4. Replace with your key

> ⚠️ **Important:** Never commit your real API key to a public repository. Use environment variables in production.
